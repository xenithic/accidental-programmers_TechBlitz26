# ClinicOS: Smart Healthcare Management System - Accidental Programmers_TechBlitz26

ClinicOS is a professional-grade, Flask-based healthcare management system designed to streamline clinic operations. It provides a premium, user-friendly interface for patients, doctors, and receptionists to manage appointments and schedules efficiently.

## 🚀 Key Features
- **Smart Appointment Scheduling:** Patients can book, reschedule, or cancel appointments in real-time.
- **Role-Based Dashboards:** Specialized views for Patients (dashboard), Doctors (patient management), and Receptionists (system overview).
- **Premium UI/UX:** Built with a modern, glassmorphic design system using CSS, Bootstrap 5, and Inter typography.
- **Secure Authentication:** Robust user authentication with password hashing and session management.
- **Real-time Data:** Powered by MongoDB for flexible and scalable data storage.

## 🛠️ Technology Stack
- **Backend:** Python / Flask
- **Frontend:** HTML5, Modern CSS, Bootstrap 5, JavaScript
- **Database:** MongoDB
- **Authentication:** Flask-Login

## ⚙️ Setup Instructions

1. **Environment Configuration:**
   - Copy `.env.example` to `.env`.
   - Update `MONGO_URI` and `SECRET_KEY` in the `.env` file.

2. **Installation:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run Application:**
   ```bash
   python app.py
   ```
