# accidental-programmers_TechBlitz26

## Setup Instructions

Before running the application, you need to set up your environment variables.
1. Copy the `.env.example` file to a new file called `.env`.
2. Update the `.env` file with your actual `MONGO_URI` and `SECRET_KEY` values:
```
MONGO_URI=your_actual_mongodb_connection_string
SECRET_KEY=your_actual_secret_key
```
3. Install the dependencies by running `pip install -r requirements.txt`.
